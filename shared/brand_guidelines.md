# UK CCTV & Home Security Brand Guidelines
## Color Palette
- Primary: #1A237E (Deep Security Blue) - Trust, Authority, Professionalism.
- Secondary: #455A64 (Steel Slate) - Technical, Hardware, Industrial.
- Accent: #FFC107 (Warning Amber) - High visibility for alerts/CTAs.
- Background: #F5F7FA (Clean Grey/White) - Modern Web aesthetics.

## Typography
- Headers: Sans-serif, Bold, High contrast.
- Body: Clean, readable geometric sans-serif.

## Visual Style
- Flat Vector Illustrations (Modern, minimal).
- High-contrast data visualizations.
- Professional, technical, and authoritative (avoiding "cartoonish" looks).
- Include "Verified by Gary Pearce" seal on technical diagrams.
